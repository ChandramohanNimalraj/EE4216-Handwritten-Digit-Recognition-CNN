\section{Discussion}

This study successfully developed and evaluated five different Convolutional Neural Network (CNN) architectures for handwritten digit recognition using the MNIST dataset. Each model was designed by modifying a single CNN parameter while maintaining the remaining network configuration unchanged. This approach enabled a systematic investigation of how architectural changes influence classification performance.

The experimental results demonstrated that all five CNN models achieved excellent handwritten digit recognition performance, with testing accuracies exceeding 97\%. Among the evaluated models, Model 3, which incorporated an additional convolution layer, achieved the highest testing accuracy of 99.11\% and the lowest number of misclassified images. The additional convolution layer improved the network's ability to extract higher-level image features, resulting in better classification performance.

The comparison between kernel sizes showed that increasing the convolution kernel from $3\times3$ to $5\times5$ produced only a minor change in classification accuracy. Although the larger kernel reduced the number of trainable parameters and training time, the standard $3\times3$ kernel provided slightly better recognition accuracy, indicating that smaller convolution kernels are well suited for extracting local features from handwritten digit images.

The optimizer comparison revealed that the Adam optimizer significantly outperformed the SGD optimizer. While SGD required less training time, it produced lower testing accuracy, higher testing loss, and a considerably larger number of misclassified images. The adaptive learning strategy employed by Adam enabled faster convergence and more effective optimization throughout the training process.

The comparison of activation functions indicated that both ReLU and Tanh provided excellent classification performance. Model 5, which used the Tanh activation function, achieved the lowest testing loss while maintaining the same testing accuracy as the baseline model. However, ReLU remains a preferred activation function for many deep learning applications because of its computational simplicity and efficient gradient propagation.

The confusion matrix and misclassification analysis further confirmed the effectiveness of the developed CNN models. Most handwritten digits were correctly classified, while only a small number of prediction errors occurred between visually similar digit classes. These misclassifications mainly resulted from ambiguous handwriting styles rather than limitations of the CNN architecture itself.

The successful prediction of custom handwritten digit images demonstrated that the trained CNN generalized well beyond the original MNIST dataset. The preprocessing pipeline, including grayscale conversion, inversion, thresholding, resizing, padding, and normalization, enabled the custom images to closely resemble the MNIST image format, allowing the CNN to classify them accurately.

Overall, the experimental results demonstrate that CNNs provide a highly effective solution for handwritten digit recognition. The comparative analysis also highlights the importance of selecting appropriate architectural parameters, including the number of convolution layers, optimizer, kernel size, and activation function, to achieve the best balance between classification accuracy, computational efficiency, and model complexity.
